# BIG DATA EZ

大数据学习项目乱炖

项目目录说明：
- data: 存放测试数据
- src: 存放代码

## 项目

Hadoop
- HdfsTest: 基本 HDFS 读写功能
- HBaseTest: 基本 HBase 读写功能
- WordCountApp: 经典 Map Reduce 案例

Spark
- HelloWorld: 入门
- ParserApp: 基本的数据清洗
- LogEtlApp: 大致的 ETL 流程
- AnalysisApp: 后续数据分析

Zookeeper
- ZkClient: Zookeeper 客户端实现基本 CRUD 功能
